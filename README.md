# Hotel Reservation Analysis

## Project Overview
This project analyzes hotel reservation data to identify patterns affecting booking cancellations and customer behavior. The analysis was conducted using Excel, RapidMiner, and Power BI to support data-driven decision-making in the hospitality industry.

---

## Objectives
- Analyze hotel booking and cancellation trends
- Identify factors affecting reservation cancellations
- Build predictive models for cancellation behavior
- Create interactive dashboards for business insights

---

## Tools & Technologies
- Power BI
- Microsoft Excel
- RapidMiner
- Data Visualization
- Predictive Modeling
- Data Cleaning & Transformation

---

## Dataset Information
- Dataset Size: 36,275 records
- Number of Attributes: 19
- Source: Kaggle

Dataset Link:  
https://www.kaggle.com/datasets/ahsan81/hotel-reservations-classification-dataset/data

---

## Project Workflow
1. Data Cleaning and Preparation in Excel
2. Data Transformation and Analysis
3. Predictive Modeling using RapidMiner
4. Dashboard Development in Power BI
5. Business Insights and Recommendations

---

## Key Insights
- Long lead times increased cancellation probability
- Certain booking channels showed higher cancellation rates
- Repeated guests had lower cancellation behavior
- Booking patterns varied across customer segments
